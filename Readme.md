Inheritance:
============

+ Inheritance is the process of acquiring data from parent class to child class
+ Single level A-->B
+ Multi-level A-->B --> C
+ Hierarchy  A-->B & A-->C
+ Multiple Inhertance A,C -->B (Not supported in JS)
+ Inheritance we can get in js by using extend key word
